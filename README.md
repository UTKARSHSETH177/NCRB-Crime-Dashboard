# NCRB Crime Dashboard (2020–2022)

## 📊 Overview
This project uses NCRB crime statistics to build an interactive Power BI dashboard.
It includes bar charts, pie charts, line charts, and KPI cards to analyze trends and compare states.

## ⚙️ Technologies
- Power BI Desktop
- Power Query (for data cleaning/unpivoting)
- DAX (for measures and KPIs)

## 📈 Visuals
- Bar Chart: Crimes by State (2022)
- Pie Chart: Crime Distribution (2022)
- Line Chart: Crime Trend (2020–2022)
- KPI Cards: Totals, averages, highest values

## 🚀 How to Use
1. Download the `.pbix` file.
2. Open it in Power BI Desktop.
3. Interact with slicers and visuals.
